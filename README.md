# scala-cli-maven-crypto-pbkdf2-hashing

## Description
A demo for pbkdf2 hashing of
a password.

## Tech stack
- maven
- scala

## Docker stack
- openjdk:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
